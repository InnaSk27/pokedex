# Pokédex — React Pair Project

A complete React application built with Vite, React Router and the native Fetch API.

## Students

- Student A
- Student B

## API used

[PokéAPI](https://pokeapi.co/) — free, no authentication required.

## Tech stack

- React 18 + Vite
- React Router DOM v6
- Native Fetch API
- CSS Modules

## How to run

```bash
npm install
npm run dev
```

## Features

- Paginated Pokémon list (Generation I)
- Detail page per Pokémon with stats, types, moves and info
- Add / remove favourites with state lifted to `App.jsx`
- Persistent navbar with favourite counter
- Loading and error states on every fetch
